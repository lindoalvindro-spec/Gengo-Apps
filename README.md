## 🛠️ CARA PENGGUNAAN APLIKASI

Ikuti langkah mudah berikut untuk mulai menerjemahkan game Anda secara instan:

### 1. Pengaturan Bahasa
* **Source (Bahasa Game):** Pilih bahasa teks asli di dalam game Anda (misalnya: *Japanese* untuk game Jepang, atau *English* untuk game Inggris).
* **Target (Hasil Terjemahan):** Pilih bahasa hasil terjemahan yang Anda inginkan (misalnya: *Indonesian*).

### 2. Pilih Area Teks Game (Select Region)
* Buka game Anda di latar belakang.
* Di aplikasi Gengo, klik tombol **`Select Region`**.
* Layar Anda akan meredup sejenak. Silakan **klik dan tahan (drag)** kursor mouse Anda untuk membuat kotak di area tempat teks dialog game muncul.
* Lepaskan klik mouse, dan area capture Anda telah tersimpan secara otomatis!

### 3. Mulai Menerjemahkan (Start Live Capture)
* **Live Translator:** Klik tombol **`Start`**. Aplikasi akan memantau teks game Anda dan menerjemahkannya secara otomatis setiap beberapa detik sekali.
* **Sesuaikan Interval:** Atur durasi interval (misal: 2 detik atau 3 detik) sesuai kecepatan munculnya teks dialog di dalam game Anda.
* **Capture 1 Kali:** Jika Anda hanya ingin menerjemahkan sekali saja tanpa otomatis, klik tombol **`Capture Once`**.

### 4. Menggunakan Jendela Transparan (Show Overlay)
* Klik tombol **`Show Overlay`**.
* Sebuah jendela transparan yang melayang akan muncul di layar Anda.
* Geser (*drag*) dan sesuaikan ukuran (*resize*) jendela transparan tersebut, lalu letakkan tepat di atas layar game Anda. Hasil terjemahan bahasa Indonesia akan muncul langsung di sana!

## 🌟 Fitur Utama Gengo
* 🔍 **OCR Otomatis (Tesseract.js):** Membaca teks gambar di layar secara presisi.
* 🌐 **Google Translate Integration:** Menerjemahkan teks secara instan ke bahasa Indonesia yang natural.
* 🖥️ **Transparent Overlay Window:** Jendela terjemahan melayang transparan di atas game untuk pengalaman bermain yang imersif.
* 💾 **Preset Game Area:** Menyimpan koordinat area kotak game Anda sehingga tidak perlu diatur ulang setiap kali membuka game.
* 📋 **History logs:** Menyimpan riwayat terjemahan sebelumnya agar bisa dibaca kembali.
