===========================================
   GENGO — Live Game Translator
   v1.0.0
===========================================

Aplikasi penerjemah teks game secara real-time.
Capture layar, OCR (Tesseract.js), terjemahkan via Google Translate.

===========================================
   CARA INSTALL
===========================================

1. Ekstrak file ZIP ke folder mana saja
2. Buka folder hasil ekstrak
3. Double-klik "Gengo.exe"
4. Selesai! Aplikasi langsung jalan.

   >> Tidak perlu install
   >> Tidak perlu setup API key
   >> Semua dependency sudah include

===========================================
   REQUIREMENT
===========================================

- Windows 10 / 11 (64-bit)
- Koneksi internet (untuk Google Translate)

===========================================
   CARA PAKAI
===========================================

1. Pilih bahasa Source (teks game) dan Target (hasil terjemahan)

2. Klik "Select Region" -> pilih layar atau window game

3. Klik & drag untuk gambar area capture (kotak dialog game)

4. Klik "Start" untuk live capture otomatis, atau "Capture Once"
   untuk 1x terjemahan

5. Atur Interval (1-10 detik) sesuai kecepatan teks game

6. Klik "Show Overlay" untuk menampilkan jendela transparan
   di atas game (bisa di-drag & di-resize)

===========================================
   FITUR
===========================================

- Live OCR & terjemahan real-time
- Overlay transparan di atas game
- History 100 terjemahan terakhir
- Simpan region preset per game
- Copy teks ke clipboard
- Always on top
- Interval capture adjustable (1-10 detik)

===========================================
   BAHASA DIDUKUNG
===========================================

Source (OCR):  Japanese, English, Chinese (Simplified/Traditional), Korean
Target (Translate): Indonesian, English, Japanese, Korean, Chinese

===========================================
   TEKNOLOGI
===========================================

- Electron
- Tesseract.js (OCR)
- Google Translate API
- HTML/CSS/JavaScript
